UliPad自动输入括号和引号

Ahui ⋅ Tagged: Linux sofeware

打开 ulipad/conf/python.acp文件,在 [autostring_append] 行之后加入

' = "!^'"
" = '!^"'
< = "!^>"
( = "!^)"
{ = "!^}"
<square> = "!^]"
